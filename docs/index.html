**** CBS C64 Budget Planner ****

In 1983 Compute Gazette published a code listing for a budget planner for the Commodore 64 written by my father Charles B. Silbergleith.

In honor of the 35th anniversary of that publication, as well as for my father's birthday, I've resurrected that code in a git repository.

TRY IT OUT

You can try out a running version of his program below. The disk image contains a sample budget which you can load. The filename is sample.bdg

NOTES

  • The keymap for the emulator isn't quite right and the * character can be entered by pressing the ] key on the standard US Keyboard.
  • In order to end data entry in the input portions of the program you'll need to enter: *END
  • If you end up in one of the subroutines and can't get out, You can return to the main menu by pressing F6
  • For a full description of the motivation, features, operation, and design of the application, please read the original article that my father wrote.
